# Contributing

Thanks for helping improve Open Resume Builder!

## Getting started

```bash
npm install
npm run dev
```

## Development tips

- Keep components small and reusable.
- Prefer DaisyUI/Tailwind utility classes.
- Run `npm run lint` before submitting.

## Pull requests

1. Create a branch from `main`.
2. Keep PRs focused and include a short summary.
3. Update docs and screenshots if UI changes.

## Reporting issues

Include steps to reproduce, expected behavior, and screenshots if relevant.
